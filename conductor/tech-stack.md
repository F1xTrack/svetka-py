# Tech Stack

## 1. Основной язык и среда
- **Язык:** Python 3.10+ (оптимально для работы с ИИ).
- **Платформа:** Windows 10/11.
- **Ускорение:** NVIDIA CUDA (для локальной обработки и видео).

## 2. Интерфейс (GUI)
- **Библиотека:** PyQt6 / PySide6.
- **Браузерный движок:** PyQt6-WebEngine (QtWebEngine) для превью тем и веб-компонентов.
- **Возможности:** Мощная система виджетов для 50+ настроек и плавающих окон.

## 3. Захват данных
- **Экран:** MSS (скорость) + OpenCV (обработка).
- **Звук:** Sounddevice / PyAudio (микрофон + системный WASAPI Loopback).
- **Обработка:** Numpy для смешивания потоков, спектрального шумоподавления, расчета RMS и векторного поиска (RAG).
- **Система:** Native Win32 API для глубокой интеграции с Windows.

## 4. Нейросети и API
- **SDK:** OpenAI SDK (совместимость с OpenAI, Gemini, Claude).
- **TTS:** Edge-TTS для высококачественного локального синтеза.
- **Схемы:** Поддержка кастомных base_url для любых API-совместимых моделей.

## 5. Архитектура
- **Асинхронность:** asyncio + qasync для интеграции цикла событий ИИ в PyQt6 GUI.
- **Контекст:** Система управления токенами и сжатия истории.
- **Хранение данных:** Модульная JSON-архитектура для управления библиотекой из 300+ prompt блоков.
