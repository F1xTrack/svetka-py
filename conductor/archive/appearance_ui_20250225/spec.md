# Specification: Appearance & Errors UI (Tab 6)

## 1. Обзор
Разработка вкладки внешнего вида и системы уведомлений с использованием браузерного движка.

## 2. Функциональные требования
- **Стиль:** Темная тема через QtWebEngine.
- **Окно Ошибок:** Слайдеры прозрачности, цвета и таймеров.
- **Управление:** Сохранение координат окна, сброс позиции.
- **Трей:** Иконка в системном трее Windows с меню быстрого доступа.
- **Интеграция:** Связь с ConfigManager и Guide Engine.

## 3. Критерии приемки
- Браузерный интерфейс корректно отображается.
- Окно ошибок сохраняет координаты.
- Трей работает.
- Покрытие тестами 95%.
