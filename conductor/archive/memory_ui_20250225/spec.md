# Specification: Memory Settings UI (Tab 5)

## 1. Обзор
Разработка графического интерфейса для управления краткосрочной и долгосрочной памятью бота.

## 2. Функциональные требования
- **Управление Контекстом:** Слайдер лимита токенов, очередь сохранения N, авто-саммаризация.
- **Управление RAG:** База знаний (путь), глубина поиска K, очистка памяти.
- **Инструменты:** Индикатор токенов (Progress Bar), экспорт/импорт памяти.
- **Интеграция:** Связь с Memory Logic и Guide Engine.

## 3. Критерии приемки
- Все элементы управления присутствуют на вкладке.
- Индикатор токенов обновляется динамически.
- Подтверждение при очистке памяти реализовано.
- Покрытие тестами 95%.
