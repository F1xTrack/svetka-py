# Specification: Vision Processing Module

## 1. Обзор
Разработка логического ядра для захвата и предварительной обработки визуальной информации.

## 2. Функциональные требования
- **Захват:** MSS и Win32 API поддержка.
- **Pipeline:** Ресайз, CV2 фильтры, детекция движения (отсечение дублей).
- **Видео:** Сборка .webm фрагментов для Gemini.
- **Приватность:** Наложение Blur-масок и проверка Blacklist окон.
- **Интеграция:** Работа с ConfigManager и передача статуса в UI.

## 3. Критерии приемки
- Кадры захватываются.
- Blur работает.
- Детекция движения работает.
- Покрытие тестами 95%.
