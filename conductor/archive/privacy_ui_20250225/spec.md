# Specification: Privacy Settings UI (Tab 7)

## 1. Обзор
Разработка графического интерфейса для управления приватностью и безопасностью пользовательских данных.

## 2. Функциональные требования
- **Фильтрация:** Чёрный список приложений по имени процесса/заголовку. Кнопка быстрого добавления.
- **Маскирование:** Статические зоны Blur на экране.
- **Передача данных:** Глобальный Offline-Only переключатель.
- **Аудит:** Журнал отправки данных и кнопка полной очистки (Wipe).
- **Интеграция:** Связь с Privacy Shield и Guide Engine.

## 3. Критерии приемки
- Чёрный список блокирует захват соответствующих окон.
- Offline-Only режим отключает все сетевые запросы.
- Очистка данных работает корректно.
- Покрытие тестами 95%.
