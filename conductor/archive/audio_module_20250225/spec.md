# Specification: Audio Processing Module

## 1. Обзор
Разработка логического ядра для захвата, микширования и генерации аудио-информации.

## 2. Функциональные требования
- **Захват:** Микрофон + системный звук (Loopback).
- **Pipeline:** Микширование в один поток, шумоподавление.
- **STT:** Гибридная поддержка (Локальный Whisper, Cloud API, Native Gemini Multimodal).
- **TTS:** Локальные движки (Edge-TTS) и облачные премиум-API.
- **Интеграция:** Передача данных громкости в UI и работа с ConfigManager.

## 3. Критерии приемки
- Бот корректно микширует звук.
- STT и TTS работают в выбранных режимах.
- Покрытие тестами 95%.
