# Specification: Proactive Brain Logic

## 1. Обзор
Разработка логического ядра для автономного принятия решений о начале диалога.

## 2. Функциональные требования
- **Сборка данных:** Объединение скриншота, аудио и метаданных в единый запрос.
- **Принятие решения:** Модель сама решает через промпт "Am I Interested?", стоит ли комментировать ситуацию.
- **Интеграция:** Прямая связь с модулями Vision, Audio и TTS.
- **Гибкость:** Управление частотой речи только через диалог с пользователем.

## 3. Критерии приемки
- Бот самостоятельно инициирует речь при изменении контекста.
- Ответы модели парсятся корректно (JSON).
- Покрытие тестами 95%.
