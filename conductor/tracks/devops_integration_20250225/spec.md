# Specification: DevOps & Integration

## 1. Обзор
Финальный этап разработки: автоматизация, сборка и интеграция с GitHub.

## 2. Функциональные требования
- **Зависимости:** Полный переход на Poetry.
- **GitHub:** Авто-создание репозитория, настройка CI/CD (тесты + релизы).
- **Сборка:** Генерация единого .exe файла через PyInstaller.
- **Релиз:** Публикация версии v0.1.0 с артефактами.

## 3. Критерии приемки
- Poetry корректно управляет средой.
- GitHub Actions проходят успешно.
- .exe файл запускается.
- Покрытие тестами 95%.
