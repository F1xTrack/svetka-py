# Svetka-py

Проактивный ИИ-компаньон для среды Windows, интегрирующий визуальный и аудиальный контекст в реальном времени. В отличие от пассивных ассистентов, система обладает автономным циклом принятия решений и вступает в диалог на основе анализа изменений в рабочем пространстве пользователя.

## Ключевые архитектурные решения

### Проактивная модель поведения
Система не требует триггерных фраз для активации. Модуль Brain Logic анализирует совокупность факторов (динамика экрана, аудиопоток, системные события) и самостоятельно инициирует взаимодействие, используя кастомизируемые шаблоны личности.

### Гибридная система восприятия
- Vision: Захват через MSS и Win32 API. Препроцессинг включает ресайз, фильтрацию через OpenCV и детектор движения (SSIM) для исключения избыточных запросов.
- Audio: Дуплексный захват (микрофон + системный Loopback). Поддержка нативной мультимодальности Gemini 2.0/2.5 (прямая передача аудиопотока) или классической схемы STT -> LLM.

### Умное управление контекстом
Реализован алгоритм динамической компрессии истории:
- При достижении лимита в 400,000 токенов запускается фоновая саммаризация.
- Механизм защиты "Context Queue" сохраняет последние 10 итераций в исходном виде для предотвращения потери актуального контекста.
- Интегрированный RAG (FAISS/ChromaDB) для долгосрочного хранения фактов о пользователе.

### Конфигурация и Кастомизация
- Свыше 50 валидируемых параметров через YAML-движок.
- Динамическая система подсказок (Guide Engine), интегрированная в UI.
- Библиотека из 100+ модульных блоков для сборки уникальных системных промптов.

## Технологический стек
- Core: Python 3.12, Asyncio
- UI: PyQt6 + QtWebEngine (гибридный мини-клиент)
- Vision: OpenCV, MSS
- Audio: Sounddevice, PyAudio, Edge-TTS
- AI Integration: OpenAI SDK (совместим с Gemini, Claude, Local LLM)

## Развертывание

### Требования
- Windows 10/11
- NVIDIA GPU с поддержкой CUDA
- Python 3.10+

### Инструкция
1. Клонирование репозитория:
   git clone https://github.com/f1xtrack/svetka-py.git

2. Установка зависимостей через Poetry:
   poetry install

3. Запуск приложения:
   poetry run python main.py

## Приватность и Безопасность
Система разработана с приоритетом локальной обработки данных:
- Режим Offline-Only: полная блокировка внешних API-вызовов.
- Blacklist окон: автоматическая приостановка захвата при обнаружении конфиденциальных приложений.
- Статичные и динамические маски (Blur) для защиты экранных данных.
